Movie World Project

I worked on Movie World project .it is a sleek, user-friendly web application designed to enhance movie discovery. Developed using React, it fetches real-time data from The Movie Database (TMDb) API to display trending films, complete with posters, ratings, and synopses. The homepage showcases a dynamic carousel of top-rated movies, while individual movie cards provide quick access to detailed information. The app's responsive design ensures a seamless experience across devices. A subscription form is also included, allowing users to receive updates on the latest movie releases. This project demonstrates proficiency in modern web development technologies and API integration.
